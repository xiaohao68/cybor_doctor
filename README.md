# 「赛博华佗」🩺 cyber-doctor

## 项目概述

本项目是一个基于多模态大语言模型的医疗健康助手，旨在帮助用户进行健康咨询、疾病诊断、文档生成等功能。通过整合文本、语音、图片等多种交互方式，提供便捷的智能健康服务。

## 功能特点

- **多模态交互**：支持文本、语音、图片等多种输入方式
- **语音对话**：语音输入语音输出，降低使用门槛
- **图片识别**：识别病历、药品说明书等图片内容
- **文档生成**：自动生成PPT和Word文档
- **联网搜索**：通过网络检索获取最新信息
- **知识库检索**：结合上传文件进行问答
- **音频生成**：支持多种方言的语音合成

## 技术栈

- Python 3.10+
- Gradio：UI界面框架
- Langchain：链式操作框架
- Transformers：预训练模型库
- Whisper：语音转文本
- edge-tts：文本转语音

## 环境配置

1. 复制 `.env.example` 为 `.env`，填写API配置

2. 创建Python环境：
```bash
conda create --name cyber-doctor python=3.10
conda activate cyber-doctor
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

## 启动项目

```bash
python app.py
```

启动后访问 http://localhost:10032

## 使用说明

- **文本对话**：直接输入问题，支持上传PDF、DOCX、图片等文件
- **语音对话**：点击"语音对话模式"按钮，使用麦克风进行语音输入
- **图片识别**：上传图片后，系统会自动识别图片内容
- **文档生成**：输入"生成PPT"或"生成Word"相关指令
- **联网搜索**：输入包含"搜索"关键词的问题
- **知识库检索**：输入包含"根据知识库"关键词的问题

## 注意事项

- 本项目仅提供健康咨询服务，不能替代专业医疗诊断
- 使用前请确保已配置有效的大模型API密钥
- 建议在网络环境良好的情况下使用联网搜索功能
